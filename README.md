# Windows 桌面倒计时工具

一个功能强大的 Windows 桌面倒计时工具，支持多任务管理、历史记录统计和直观的用户界面。

## 功能特点

- 📅 **多任务管理**：支持创建多个倒计时任务，可随时切换
- ⏰ **大型倒计时显示**：清晰的数字显示，方便查看剩余时间
- 📊 **历史记录**：自动记录完成的任务，按日期分组显示，计算每天总学习时间
- 🔔 **结束提醒**：倒计时结束时发出声音提醒并弹出通知
- 🎨 **现代化界面**：使用 CustomTkinter 构建的深色主题界面
- 💾 **数据持久化**：任务和历史记录自动保存到本地文件
- ✏️ **任务编辑**：支持编辑和删除倒计时任务
- 🔄 **状态保持**：程序关闭后重新打开可恢复倒计时状态
- 🎯 **后台运行**：关闭界面后程序可在系统托盘继续运行倒计时

## 技术栈

- Python 3.8+
- CustomTkinter (现代化 GUI 库)
- threading (多线程支持)
- json (数据存储)
- pystray (系统托盘支持)
- Pillow (图像处理)
- plyer (桌面通知)

## 安装方法

### 方法 1：直接运行可执行文件（推荐）

1. 从 `dist` 目录中找到 `CountdownTool.exe`
2. 直接双击运行即可，无需安装 Python 环境

### 方法 2：从源代码运行

1. 确保已安装 Python 3.8 或更高版本
2. 安装依赖：
   ```bash
   pip install customtkinter pystray Pillow plyer
   ```
3. 运行主程序：
   ```bash
   python CountdownTool.py
   ```

## 使用指南

### 添加倒计时任务
1. 点击左侧边栏的"添加倒计时"按钮
2. 输入任务名称和时长（小时、分钟、秒）
3. 点击"确认添加"按钮

### 开始倒计时
1. 在任务列表中选择一个任务
2. 点击右侧的"开始"按钮
3. 倒计时开始，可随时点击"暂停"或"重置"

### 编辑倒计时任务
1. 在任务列表中找到要编辑的任务
2. 点击该任务右侧的"编辑"按钮
3. 修改任务名称或时长
4. 点击"确认修改"按钮

### 删除倒计时任务
1. 在任务列表中找到要删除的任务
2. 点击该任务右侧的"删除"按钮
3. 在确认对话框中点击"是"确认删除

### 后台运行（新增）
- **最小化到托盘**：当倒计时正在运行时，点击窗口右上角的关闭按钮，程序会最小化到系统托盘继续运行
- **恢复窗口**：在系统托盘图标上右键点击，选择"显示窗口"即可恢复主界面
- **完全退出**：在系统托盘图标上右键点击，选择"退出程序"即可完全关闭程序
- **桌面通知**：倒计时结束时，如果窗口已隐藏，会显示桌面通知提醒

### 查看历史记录
1. 点击左侧边栏的"查看历史记录"按钮
2. 查看按日期分组的历史任务记录
3. 每条记录显示任务名称、时长和完成时间
4. 每天的总学习时间会自动计算并显示

## 文件说明

- `CountdownTool.py` - 主程序源代码
- `CountdownTool.exe` - Windows 可执行文件（位于 dist 目录）
- `countdown_config.json` - 任务配置文件（程序运行后自动生成）
- `countdown_history.json` - 历史记录文件（程序运行后自动生成）

## 打包成可执行文件

如果需要重新打包程序，可以使用 PyInstaller：

```bash
pip install pyinstaller
python -m PyInstaller --onefile --windowed --name CountdownTool CountdownTool.py
```

生成的可执行文件将位于 `dist` 目录中。
